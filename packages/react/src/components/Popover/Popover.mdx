## Overview

**Note: this component is currently experimental and does not follow semver.**

A popover is a UI element that _pops over_ or renders on top of other UI on the
page. It's a component "primitive" meaning it can be used to help make more
complex components that utilize this behavior. It is often styled as what we'd
traditionally consider a tooltip.

## Usage 

```jsx
import { popover } from 'carbon-components-react';

function MyPopover() {
  return (
    <TriggeringUiElement/>
    <Popover 
      open={open} 
      align={top} 
      relative>
      <PopoverContent className="custom-classname"/>
    </Popover>
  )
}
```
